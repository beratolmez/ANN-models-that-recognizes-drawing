Django-based drawing recognition application with Artificial Neural Networks.

🐍 Downloading Model Files (ml_models)

🚨 Note: Due to the large total size of the models, they are not available in the GitHub repository. You must download them from the link below and place them in the ml_models/ folder:

🔗 <a href="https://drive.google.com/drive/folders/14ZbfJlWprTypAcgFW5Q17Ai5fEKzvhma?usp=sharing">ml_models/ Download (Google Drive)</a>

📂 Post-Installation: Place the downloaded ml_models/ folder under the my_apps/ directory in the project directory. You can also add and run your own models here.
